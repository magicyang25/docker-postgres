# PostgreSQL DB in Docker

## What's in this repo

Dockerized PostgreSQL DB with the following versions

- 10.3
- 11.4

## Why

To save the time and avoid troubles of managing multiple versions of PostgreSQL running on a machine.

## How to use

### 10.3

```bash
# to start
$ auto/run-10-3

# to stop
$ auto/down-10-3
```

### 11.4

```bash
# to start
$ auto/run-11-4

# to stop
$ auto/down-11-4
```
